### ProjectE: Генерация аниме-персонажей
End-to-End ML продукт с двумя каналами взаимодействия для создания уникальных изображений

**Технологии:** Python, TensorFlow, Flask, Docker, Telegram API

**Ссылка:** [GitHub](https://github.com/OczyDimon/ProjectE)

- Обучил модель GAN, используя открытый датасет и TF/Keras
- Разработал end-to-end систему генерации изображений на основе GAN
- Реализовал веб-интерфейс и Telegram-бота для взаимодействия с моделью
- Настроил REST API с помощью Flask и деплой на сервер

**Результат:** работающее приложение с двумя каналами доступа для генерации узнаваемых лиц

### CoinRecognition: Система распознавания монет 
Система идентификации монет с использованием методов обработки изображений и сравнения с шаблонами

**Технологии:** Python, OpenCV, Scikit-learn, NumPy

**Ссылка:** [GitHub](https://github.com/OczyDimon/coins_recognition)

- Реализовал алгоритм детекции монет на изображениях с помощью HoughCircles
- Создал систему сравнения с шаблонами через template matching с поворотами
- Разработал классификатор номиналов на основе максимального скора совпадения

**Результат:** точное определение номиналов монет Банка России 

### Sudoku Solver: Распознавание и решение судоку
Компьютерное зрение и алгоритмическое решение задачи с веб-интерфейсом

**Технологии:** Python, OpenCV, TensorFlow, Flask, Backtracking

**Ссылка:** [GitHub](https://github.com/OczyDimon/sudoku_solver)

- Разработал веб-приложение на Flask для загрузки и обработки изображений
- Реализовал компьютерное зрение для детекции сетки судоку и классификации цифр через CNN
- Создал алгоритм решения на основе backtracking с валидацией решений

**Результат:** система end-to-end: от загрузки фото до готового решения
